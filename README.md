# How to Run
- Install Android Studio
- Install Flutter SDK
- Install Android SDK
- Open csci498proj1 folder in Android Studio
- Open ottov1 folder
- Select Android device to deploy app on
- Clone repo
- Run command "export PATH="/Applications/flutter/bin:$PATH"
- Run command "flutter run --no-sound-null-safety"
